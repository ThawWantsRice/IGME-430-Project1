POST Endpoints

/addPokemon

POST /addPokemon

Query Parameters

Parameter Required Description
name Yes Full Pokemon Name
type Yes Full Pokemon Type
weight Yes Pokemon Weight
Height Yes Pokemon Height
IMG No IMG URL http://

Example:

Request: /addPokemon

Body: {name: 'name', type: '[normal]', weight: '1', height: '1', img:''}

Result: {"pokemon": {"id": 152,"num": "152","name": "name","type": "Normal","weight": "1","height": "1","img": "","weaknesses": []}}

Request: /addPokemon

Body: {name: '', type: '[normal]', weight: '1', height: '1', img:''}

Result:{"message": "Missing required fields","id": "missingParams"}
